# Прогнозирование коэффициента восстановления золота

## Задача
Построение модели для предсказания эффективности обогащения золотосодержащей руды на промышленном предприятии. Целевая метрика — симметричное среднее абсолютное процентное отклонение (sMAPE).

## Подход
- Восстановление пропусков методом линейной интерполяции (данные индексируются временем, соседние наблюдения коррелируют)
- Удаление признаков, недоступных в тестовой выборке (кроме целевых переменных `rougher.output.recovery` и `final.output.recovery`)
- Фильтрация аномальных значений концентрации (<20) как потенциальных ошибок измерений
- Верификация расчёта целевых признаков по формуле эффективности обогащения:
  $$Recovery = (C * (F - T)) / (F * (C - T)) * 100%$$

  где C — концентрация в концентрате, F — в сырье, T — в хвостах

## Модели и результаты
- Базовая линейная регрессия: sMAPE ≈ 9.99
- Случайный лес (оптимальные параметры):
- `n_estimators = 40`
- `max_depth = 7`
- Итоговая метрика на тесте: **sMAPE ≈ 8.7%**

Анализ важности признаков выявил ключевые факторы:
- Концентрация серебра в сырье (`rougher.input.feed_ag`)
- Дозировка сульфата на этапе первичной очистки (`primary_cleaner.input.sulfate`)
