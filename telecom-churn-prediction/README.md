## Прогнозирование оттока клиентов телеком-оператора

**Задача:** Разработка модели для снижения оттока клиентов.

**Технологии:** Python, Pandas, Scikit-learn, LightGBM, Matplotlib

**Ключевые результаты:**
- Обработаны 7043 записи клиентов
- Проведён полный цикл подготовки данных: обработка пропусков, создание признаков (продолжительность контракта), борьба с дисбалансом классов.
- Обучены и сравнены 3 модели (логистическая регрессия, случайный лес, LightGBM).
- Достигнута метрика ROC-AUC 0.75
- Выявлены ключевые факторы оттока: ежемесячные траты >79 ₽, месячный контракт
