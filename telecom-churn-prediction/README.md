## Прогнозирование оттока клиентов телеком-оператора

**Задача:** Разработка модели для снижения оттока клиентов.

**Технологии:** Python, pandas, scikit-learn, LightGBM

**Ключевые результаты:**
- Обработаны 7043 записи клиентов
- Достигнута метрика ROC-AUC 0.75
- Выявлены факторы оттока: ежемесячные траты >79 ₽, месячный контракт
