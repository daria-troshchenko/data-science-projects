# **Анализ отзывов о винах (Wine Tasting)**

## **Задача**
Анализ датасета из 130 тыс. отзывов о винах для выявления закономерностей: связь цены и рейтинга, региональные особенности, анализ текстовых описаний.

Данные для анализа взяты из платформы [kaggle](https://www.kaggle.com/datasets/mysarahmadbhat/wine-tasting?resource=download).

## **Подход**
- Обработка пропусков:
  - Удаление 7.68% дубликатов
  - Восстановление цены медианным значением по стране и количеству баллов (учёт мультиколлинеарности с другими признаками)
  - Замена пропусков в категориальных признаках на маркер «unknown»
- Создание новых признаков:
  - Качество вина («good» / «better» / «best») на основе баллов
  - Ценовой сегмент («our price» / «affordable» / «expensive» / «luxury»)
  - «Старый/Новый свет» (Европа vs остальной мир)
  - Год производства из названия вина
- Лемматизация текстовых описаний с использованием NLTK и библиотеки `pymystem3`

## **Ключевые результаты**
- Умеренная корреляция между баллами и ценой (**0.46**)
- Лидеры по среднему баллу:
  - Португалия (регион Madeira) — 94 балла
  - Австрия (регион Südburgenland) — 94 балла
- Лидеры по средней цене:
  - Швейцария — `$72`
  - Англия — `$53`
- Распределение цен: медиана - `$14`, 75-й перцентиль - `$25`.

## **Технологии**
- Python (pandas, numpy, nltk, pymystem3, seaborn)
- Статистический анализ (корреляционная матрица, группировка по признакам)
- Визуализация распределений и зависимостей
