# Анализ культурных событий Москвы (kudago.com)

## Задача
Сбор и анализ данных о предстоящих культурных мероприятиях в Москве для выявления закономерностей: ценовые сегменты, возрастные ограничения, сезонность и корреляция между ценой и вовлечённостью (лайками).

## Подход
- Веб-скрейпинг 1000+ событий с платформы kudago.com с обработкой динамической загрузки контента через Selenium
- Парсинг структурированных данных: название, описание, цена, возрастные ограничения, дата, место проведения, количество лайков
- Создание новых признаков:
  - Минимальная/максимальная/средняя цена (обработка текстовых диапазонов: «от 1000 до 2700»)
  - Возрастная категория («Для детей», «Для всей семьи», «Для взрослых»)
  - Формат мероприятия («Офлайн» / «Онлайн»)
  - Период события («Постоянно» / «Временно»)

## Ключевые результаты
- Медианная цена мероприятий: **1400 ₽**
- 90% событий имеют временный характер (не постоянные выставки/площадки)
- Максимальная вовлечённость наблюдается при цене до **4000 ₽**
- Сезонность: пик публикаций приходится на 2021 год (309 мероприятий), что составляет ~44% от выборки

## Технологии
- Python (Selenium, BeautifulSoup, pandas, re)
- Визуализация (Plotly для интерактивных графиков)
- Работа с текстовыми данными и регулярными выражениями
